# Databases and ORMs

## Sqlx

[sqlx (github)]( https://github.com/launchbadge/sqlx ): the Rust SQL Toolkit. An async, pure Rust SQL crate featuring compile-time checked queries without a DSL. Supports PostgreSQL, MySQL, SQLite, and MSSQL.
