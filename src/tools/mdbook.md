# mdBook

[mdBook]( https://github.com/rust-lang/mdBook ): a utility to create modern online books from Markdown files.

```bash
cargo install mdbook
cargo install mdbook-hide  # optional plugin; many others exist
```

```bash
mdbook serve --open
```
