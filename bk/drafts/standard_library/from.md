Traits for conversions between types: [`AsRef`](https://doc.rust-lang.org/std/convert/trait.AsRef.html), [`From`](https://doc.rust-lang.org/std/convert/trait.From.html)⮳ and others.

## Deref

[![std][c~std~docs~badge]][c~std~docs]

The `Deref` trait allows you to customize how your types behave when dereferenced, similar to how references work. This is useful for smart pointers and other types that need to behave like references.
